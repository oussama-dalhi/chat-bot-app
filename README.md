# React Chatbot UI 💬

A modern chatbot interface built with React and JavaScript featuring a clean dark theme and dynamic messaging.

## Features

- Dynamic chat messages
- User & bot avatars
- Auto-scrolling chat
- Responsive dark UI
- Simple chatbot logic
- Component-based structure

## Tech Stack

- React
- JavaScript
- CSS

## Installation

```bash
git clone https://github.com/oussama-dalhi/chat-bot-app.git

cd chatbot

npm install

npm run dev
```
